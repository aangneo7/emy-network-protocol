# Emy Network ($EMY) Technical Specifications

## Architecture
Emy Network is a sovereign Layer 1 built using the @CNPYNetwork infrastructure.

### Consensus Mechanism
- **Engine**: NestBFT (Custom BFT Consensus)
- **Security Extensions**: Verifiable Delay Functions (VDFs) to prevent long-range attacks.
- **Validator Selection**: Proof-of-Age (PoA) for fair decentralization.

### Network Parameters
- **Chain ID**: 2026 (Provisional)
- **Target Block Time**: 2 Seconds
- **Finality**: Enterprise-grade (Instant)
